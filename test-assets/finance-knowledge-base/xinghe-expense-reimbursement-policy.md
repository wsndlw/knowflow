# 星河智造科技有限公司费用报销管理办法

版本：V1.0
适用范围：星河智造科技有限公司全体员工。

## 一、目的

为规范星河智造科技有限公司费用报销行为，明确报销资料、审批流程和付款要求。

## 二、基本原则

1. 费用必须真实发生，且与公司经营活动相关。
2. 发票抬头必须为“星河智造科技有限公司”。
3. 申请人对业务真实性、金额准确性和附件完整性负责。
4. 未完成审批的报销单不得付款。

## 三、报销流程

1. 申请人填写报销单，选择费用类型、部门和项目编号。
2. 上传发票、合同、行程单、验收证明等资料。
3. 部门负责人审核业务真实性和预算占用。
4. 财务共享中心审核发票合规性、金额一致性和重复报销风险。
5. 超过权限的报销单提交财务负责人或总经理审批。
6. 出纳按已审批报销单付款，会计月末入账归档。

## 四、不予报销情形

与公司业务无关的个人消费、发票信息不符合要求、未经事前审批的超标准费用不予报销。
